Chaine de caractères
====================

Les chaines de caractères sont un type de données qui permet de manipuler des textes. Une variable en Python qui
contient un texte est de type *chaine de caractères* ou **str** qui est une abbréviation  de **string**.

En python le type **str** permet:

- de modifier une chaine de caractères
- de parcourir une chaine de caractères
- d'effectuer des opérations sur les chaines de caractères.

Nous allons voir différentes situations pour manipuler des chaines de caractères.
